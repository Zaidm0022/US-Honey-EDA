# Honey Production Data Analysis

This project contains an Exploratory Data Analysis (EDA) of honey production in the United States from 1995 to 2021. The analysis is performed using Python in a Jupyter Notebook.

## Dataset
The dataset used in this project is `csv dataset.csv` (or the respective CSV data file in the root directory), which includes the following information across various states and years:
- `state`: Name of the state
- `colonies_number`: Number of honey-producing colonies
- `yield_per_colony`: Honey yield per colony
- `production`: Total production
- `stocks`: Stocks held by producers
- `average_price`: Average price
- `value_of_production`: Total value of production
- `year`: Year of the record (1995 - 2021)

## Tools and Libraries Used
- **Python 3**
- **Jupyter Notebook** (`Untitled1.ipynb`)
- **Pandas**: For data manipulation and analysis
- **NumPy**: For numerical computations
- **Matplotlib & Seaborn**: For data visualization

## Project Structure
- `Untitled1.ipynb`: The main Jupyter Notebook containing the code for data loading, cleaning, and exploratory data analysis.
- `csv dataset.csv`: The dataset file used for the analysis.

## Analysis Steps Included
1. **Data Loading**: Importing the honey production dataset using Pandas.
2. **Data Inspection**: Checking the shape, data types, and first few rows using `.head()` and `.tail()`.
3. **Data Cleaning**: Checking for missing (`null`) values, identifying duplicate rows, and removing unnecessary columns (like the autogenerated `Unnamed: 0` index column).
4. **Exploratory Data Analysis**: Exploring the data functionally and visually.

## How to Run
1. Ensure you have Python installed along with Jupyter Notebook.
2. Install the required libraries if you haven't already:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook from the command line:
   ```bash
   jupyter notebook Untitled1.ipynb
   ```
4. Run the notebook cells sequentially to reproduce the analysis.
